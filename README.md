vue2 + element 前后端分离抗疫网站
项目功能：
1.首页文章专栏，评论,点赞
2.视频专栏
3.图片专栏
4.心愿专栏
5.个人中心
6.好友添加
7.发布文章
8.好友点赞文章，
9.echarts图表展示好友文章
10.意见反馈，发送邮件
11.编辑个人资料，修改个人信息，个人头像
